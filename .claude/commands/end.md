# Session beenden

## Vorgehen

1. **Pruefen ob Handoff aktuell ist:**
   - Wenn substanzielle Aenderungen passiert sind: `/handoff` ausfuehren
   - Sonst: kurz dem User sagen "Handoff ist noch aktuell"
2. **Memory-Session schliessen:** `mcp__memory__memory_session_end`
3. **Letzter Status:**
   - Offene/uncommittete Aenderungen?
   - Was wird die naechste Session zuerst sehen?

Keine ausfuehrliche Zusammenfassung — der Handoff erledigt das.
